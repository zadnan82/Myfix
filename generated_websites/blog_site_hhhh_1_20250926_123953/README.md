# Blog Site Template with Admin

Complete blog website template with admin panel, built using SEVDO frontend compiler

Generated with SEVDO Framework

## Quick Start

### Frontend
```bash
cd frontend
npm install
npm start
```

### Backend  
```bash
cd backend
pip install -r requirements.txt
uvicorn main:app --reload
```
